Scotty3D Application

Overview

Scotty3D is a C++ application written throughout Carnegie Mellon's Computer Graphics course.

Intent

Process

Scotty3D also features ray tracing algorithms to render the scenes created within the application. The algorithm uses a Bounding Volume Hierarchy structure to accelerate the collision tests for scenes with large numbers of polygons.
